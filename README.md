# How to run?

## Pre-requisites
- Web Browser (Recommended: Chrome)
- Passion for World Wide Web, its complexity and its simplicity ;)

## Run
- `clone` or `download` the code
- Open `index.html`
- Done - You should see the web browser displaying the Page
